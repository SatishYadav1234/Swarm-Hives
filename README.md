````markdown
# PyCluster 🕸️

**Seamless Distributed Computing for Local Area Networks**

PyCluster is a specialized Python toolkit that lets you effortlessly distribute and execute Python functions across multiple machines on the same LAN. By completely bypassing cloud dependencies and complex infrastructure setups, PyCluster delivers a fast, secure, and localized parallel computing environment right out of the box.

## ✨ Core Capabilities

- **🔐 Robust Encryption**: Secures all cross-device data traffic using Fernet cryptography.
- **🔑 OTP-Based Access**: Workers authenticate securely using a unique one-time password generated by the host machine.
- **🎯 Frictionless Decorators**: Tag any function with `@remote()` to instantly ship it to a remote node for execution.
- **📁 Native File Transfers**: Built-in protocols for moving large files in manageable chunks.
- **🔄 Modern Asynchronous Design**: Built from the ground up to support Python's `async/await` syntax.
- **📊 Intelligent Load Balancing**: Tasks are automatically routed to available workers to optimize execution time.
- **🛠 Handy CLI**: Manage your cluster directly from the terminal with intuitive commands.
- **🌐 100% Offline**: Requires zero internet connectivity—everything stays strictly within your local network.
- **🔒 On-the-Fly Key Exchange**: Encryption keys are dynamically generated and shared during the initial connection handshake.
````

### Core Workflow

#### 1\. Spin up the Dispatcher (Host)

```bash
# Launch the host on the default port (8888)
pycluster host

# Alternatively, define a specific port
pycluster host --port 9999
```

*Note: The terminal will output a One-Time Password (OTP). Keep this handy for your worker nodes.*

#### 2\. Attach Worker Nodes

Execute this on other computers connected to the same network:

```bash
# Link a worker to the host (swap with your actual IP and generated OTP)
pycluster join --host 192.168.1.100 --key ABC12345

# Optionally assign a custom identifier to the worker
pycluster join --host 192.168.1.100 --key ABC12345 --worker-id processing-node-alpha
```

#### 3\. Execute Distributed Code

```python
from pycluster import remote, set_host, Host
import asyncio

# Initialize the host controller
master_node = Host(port=8888)
set_host(master_node)

# Flag functions for network distribution
@remote()
def get_fibonacci(sequence_length):
    """Computes the Fibonacci sequence remotely"""
    if sequence_length <= 1:
        return sequence_length
    x, y = 0, 1
    for _ in range(2, sequence_length + 1):
        x, y = y, x + y
    return y

@remote(computer="processing-node-alpha")  # Route to a designated machine
def intensive_task(iterations):
    """A CPU-heavy task constrained to a specific worker"""
    total = 0
    for i in range(iterations):
        total += i ** 2
    return total

# Trigger the remote functions
async def run_cluster():
    # Tasks are dispatched across the network
    fib_result = await get_fibonacci(100)
    math_result = await intensive_task(10000)
    
    print(f"Fibonacci Output: {fib_result}")
    print(f"Intensive Math Output: {math_result}")

# Execute
asyncio.run(run_cluster())
```

## 📖 Deep Dive Documentation

### System Layout

The PyCluster ecosystem relies on three foundational pillars:

1.  **Dispatcher (Host)**: The brain of the operation that tracks available nodes and delegates tasks.
2.  **Executor Nodes (Workers)**: The connected devices that accept workloads, process them, and return results.
3.  **API Library**: The Python interface used to define routing logic and tag remote functions.

### Security Posture

We take local network security seriously:

1.  **OTP Gates**: Connections require a temporary passcode created by the host.
2.  **Dynamic Handshakes**: Cryptographic keys are exchanged safely during connection.
3.  **Ciphered Traffic**: Every byte of data is encrypted via AES-128 (Fernet).
4.  **Zero Persistence**: Keys are wiped when the session ends; nothing is stored permanently.

### Environment Configuration

Customize PyCluster's behavior via environment variables:

```bash
# Network Bindings
```

### Module Reference

#### Interacting with the Host

```python
from pycluster import Host

# Boot up the dispatcher
master = Host(port=8888)
await master.start()

# Retrieve the current session's OTP
passcode = master.get_otp()

# Push a task without decorators
outcome = await master.execute_task("job_123", function_name, args, kwargs)

# Poll the cluster status
active_nodes = master.get_workers_info()
```

#### Interacting with a Worker

```python
from pycluster import Worker

# Initialize and connect a node
node = Worker(
    worker_id="processing-node-alpha",
    host="192.168.1.100",
    port=8888,
    otp="ABC12345"
)
await node.start()
```

#### The `@remote` API

```python
from pycluster import remote

@remote()  # Dispatches to the next available node
def merge_data(a, b):
    return a + b

@remote(computer="processing-node-alpha")  # Forces execution on a strict target
def secure_process(data):
    return process(data)
```

```python
from pycluster.network import FileTransfer

# Push payloads to a worker node
uploader = FileTransfer(socket_manager)

    "massive_dataset.csv",
```

### Bring Your Own Keys (BYOK)

```python
from pycluster.network import EncryptionManager

# Inject custom cryptography
custom_key = EncryptionManager.generate_key()
cipher = EncryptionManager(custom_key)
```

### Fault Tolerance

```python
from pycluster import remote

@remote()
def volatile_task():
    try:
        # Code that might fail
        return dangerous_computation()
    except Exception as err:
        # Failsafes to prevent worker crashes
        return f"Execution Failed: {err}"

# Exceptions are cleanly routed back to the host dispatcher
final_data = await volatile_task()
```

## 🚀 Deployment Crash Course

### Scenario A: The Basics

```python
# Terminal 1: run `pycluster host --port 8888`
# Note the OTP!

# Terminal 2 (Python code):
from pycluster import remote, set_host, Host
import asyncio

master = Host(port=8888)
set_host(master)

@remote()
def multiplier(x, y):
    return x * y

async def execute():
    val = await multiplier(5, 10)
    print(f"Calculated: {val}")  # Output: Calculated: 50

asyncio.run(execute())
```
